One of the greatest powers of MDX is the ability to drop interactive React components anywhere inside your Markdown content. The `VideoEmbed` component brings this to life for video content ‚Äî whether you want to embed a YouTube tutorial, a YouTube Shorts clip, or any other iframe-compatible video source.

---

## How It Works

The `VideoEmbed` component is globally registered in this project's MDX component registry, which means you can use it in **any** blog post or documentation page without needing to import it.

It automatically resolves the following URL formats into a proper iframe embed:
- ‚úÖ Full YouTube URL: `https://www.youtube.com/watch?v=VIDEO_ID`
- ‚úÖ Short YouTube URL: `https://youtu.be/VIDEO_ID`
- ‚úÖ YouTube Shorts: `https://www.youtube.com/shorts/VIDEO_ID`
- ‚úÖ Raw embed URL: `https://www.youtube.com/embed/VIDEO_ID`
- ‚úÖ Any `iframe`-compatible URL (Vimeo, Loom, etc.)

---

## Basic Usage

Drop the component directly inside your `.mdx` file:

```mdx
<VideoEmbed src="https://www.youtube.com/watch?v=dQw4w9WgXcQ" />
```

<VideoEmbed
  src="https://www.youtube.com/watch?v=dQw4w9WgXcQ"
  title="Rick Astley - Never Gonna Give You Up"
  caption="A classic. The VideoEmbed component automatically resolves this YouTube URL."
/>

---

## With a Custom Aspect Ratio

By default, videos render at the classic **16:9** widescreen ratio. You can change this with the `aspect` prop. Supported values are `16/9`, `4/3`, `1/1`, and `9/16` (for vertical Shorts-style content).

```mdx
<VideoEmbed
  src="https://youtu.be/dQw4w9WgXcQ"
  aspect="4/3"
  caption="Using a 4:3 aspect ratio"
/>
```

---

## All Supported Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `src` | `string` | ‚Äî | YouTube URL, youtu.be link, or any iframe URL. **Required.** |
| `title` | `string` | `"Embedded video"` | Accessibility title for the iframe. |
| `aspect` | `string` | `"16/9"` | Aspect ratio: `16/9`, `4/3`, `1/1`, or `9/16`. |
| `caption` | `string` | ‚Äî | Optional caption displayed below the video. |

---

## Using a youtu.be Short Link

```mdx
<VideoEmbed
  src="https://youtu.be/dQw4w9WgXcQ"
  caption="Short links work too!"
/>
```

<Callout type="tip">
You can copy any YouTube video URL directly from your browser address bar or the YouTube share button ‚Äî the `VideoEmbed` component will parse it automatically!
</Callout>

<Callout type="note">
Videos are loaded **lazily** using `loading="lazy"` on the iframe, so they won't slow down your page load if you have multiple embeds on the same post.
</Callout>
