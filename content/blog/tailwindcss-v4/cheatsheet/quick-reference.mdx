# Migration Guides

## Migrating from Tailwind CSS v3 to v4

| Change | v3 | v4 |
|--------|-----|-----|
| Import | `@tailwind base; @tailwind components; @tailwind utilities;` | `@import "tailwindcss";` |
| Config file | `tailwind.config.js` | `styles.css` with `@theme` |
| Custom utilities | `addUtilities()` in JS | `@utility` in CSS |
| Custom variants | `addVariant()` in JS | `@variant` in CSS |
| Container queries | `@tailwindcss/container-queries` plugin | Built-in with `@` prefix |
| Theme values | `theme.extend.colors` | `--color-*` in `@theme` |

### Step-by-Step Migration

```bash
# 1. Update package
npm install tailwindcss@latest @tailwindcss/vite@latest

# 2. Replace imports in CSS
# OLD: @tailwind base; @tailwind components; @tailwind utilities;
# NEW: @import "tailwindcss";

# 3. Move config to CSS
# OLD: tailwind.config.js with theme.extend
# NEW: @theme { --color-brand: #3b82f6; } in CSS

# 4. Update custom utilities
# OLD: plugin(function({ addUtilities }) { ... })
# NEW: @utility custom { ... }

# 5. Update Vite config
# Remove: import tailwindcss from '@tailwindcss/postcss'
# Use:    import tailwindcss from '@tailwindcss/vite'
```
