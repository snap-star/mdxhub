### 1.1 Installation with Vite + React 19

```bash
# Create a new Vite project with React 19
npm create vite@latest my-app -- --template react-ts
cd my-app

# Install Tailwind CSS v4 (PostCSS plugin)
npm install tailwindcss@latest @tailwindcss/postcss@latest postcss

# For Vite projects, use the Vite plugin instead (recommended)

npm install react@latest react-dom@latest
npm install -D @types/react@latest @types/react-dom@latest
```

### 1.2 Configuration: The CSS-First Way

**File: `src/styles.css`** — Your configuration AND styles in one file

```css
/* ============================================
   Tailwind CSS v4 — CSS-First Configuration
   ============================================ */

/* Import Tailwind CSS — replaces @tailwind directives */
@import "tailwindcss";

/* ============================================
   THEME CUSTOMIZATION
   Define your design tokens directly in CSS
   ============================================ */

@theme {
  /* Custom colors */
  --color-brand: #0ea5e9;
  --color-brand-dark: #0284c7;
  --color-brand-light: #7dd3fc;
  
  /* Custom fonts */
  --font-sans: "Inter", ui-sans-serif, system-ui, sans-serif;
  --font-mono: "Fira Code", ui-monospace, monospace;
  
  /* Custom spacing scale */
  --spacing-18: 4.5rem;
  --spacing-88: 22rem;
  
  /* Custom breakpoints */
  --breakpoint-3xl: 120rem;
  
  /* Custom animations */
  --animate-fade-in: fade-in 0.3s ease-out;
  --animate-slide-up: slide-up 0.4s ease-out;
}

/* Define the keyframes for custom animations */
@keyframes fade-in {
  from { opacity: 0; }
  to { opacity: 1; }
}

@keyframes slide-up {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}

/* ============================================
   CUSTOM UTILITIES
   Define your own utility classes
   ============================================ */

@utility text-balance {
  text-wrap: balance;
}

@utility text-pretty {
  text-wrap: pretty;
}

@utility container-tight {
  margin-inline: auto;
  padding-inline: 1rem;
  max-width: 65ch;
}

/* ============================================
   BASE STYLES (replaces @layer base)
   ============================================ */

body {
  font-family: var(--font-sans);
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

/* React 19: Prevent layout shifts with Suspense boundaries */
[data-suspense-boundary] {
  contain: layout style;
}
```

## Chapter 2: The Utility-First Methodology

<Badge variant="green">Junior</Badge>

### 2.1 What Is Utility-First?

**Utility-first CSS** means building interfaces using small, single-purpose utility classes rather than writing custom CSS. Instead of:

```css
/* Traditional approach: custom CSS */
.alert {
  padding: 1rem;
  background-color: #fee2e2;
  border: 1px solid #fecaca;
  border-radius: 0.5rem;
  color: #991b1b;
}
```

You compose utilities directly in your HTML/JSX:

```tsx
// Utility-first approach: compose in JSX
<div className="rounded-lg border border-red-200 bg-red-50 p-4 text-red-800">
  This is an alert message.
</div>
```

### 2.2 Core Utility Categories

Tailwind v4 organizes utilities into these categories:

#### Layout

```tsx
/* === Container === */
<div className="mx-auto max-w-7xl px-4 sm:px-6 lg:px-8">
  {/* mx-auto: center horizontally */}
  {/* max-w-7xl: max-width 80rem */}
  {/* px-4: padding-x 1rem, sm:px-6: 1.5rem on sm+, lg:px-8: 2rem on lg+ */}
</div>

/* === Display === */
<div className="flex">           {/* display: flex */}
<div className="grid">           {/* display: grid */}
<div className="hidden">         {/* display: none */}
<div className="block">          {/* display: block */}
<div className="inline-flex">    {/* display: inline-flex */}

/* === Position === */
<div className="relative">       {/* position: relative */}
<div className="absolute top-0 left-0"> {/* position: absolute, top: 0, left: 0 */}
<div className="fixed inset-0">  {/* position: fixed, inset: 0 */}
<div className="sticky top-0">   {/* position: sticky, top: 0 */}
```

#### Flexbox & Grid

```tsx
/* === Flexbox === */
<div className="flex items-center justify-between gap-4">
  {/* flex: display flex */}
  {/* items-center: align-items center */}
  {/* justify-between: justify-content space-between */}
  {/* gap-4: gap 1rem */}
  
  <div className="flex-1">       {/* flex: 1 1 0% */}
  <div className="flex-shrink-0"> {/* flex-shrink: 0 */}
  <div className="grow">         {/* flex-grow: 1 */}
</div>

<div className="flex flex-col items-start">
  {/* flex-col: flex-direction column */}
  {/* items-start: align-items flex-start */}
</div>

/* === Grid === */
<div className="grid grid-cols-3 gap-4">
  {/* grid-cols-3: 3 equal columns */}
  
  <div className="col-span-2">   {/* span 2 columns */}
  <div className="row-span-2">   {/* span 2 rows */}
  <div className="col-start-2">  {/* start at column 2 */}
</div>

<div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-4 gap-6">
  {/* Responsive: 1 col mobile, 2 col tablet, 4 col desktop */}
</div>
```

#### Spacing (Padding & Margin)

```tsx
/* === Padding === */
<div className="p-4">            {/* padding: 1rem (all sides) */}
<div className="px-4 py-2">      {/* padding-x: 1rem, padding-y: 0.5rem */}
<div className="pt-8 pr-4 pb-6 pl-2"> {/* individual sides */}

/* === Margin === */
<div className="m-4">            {/* margin: 1rem */}
<div className="mx-auto">        {/* margin-x: auto (center) */}
<div className="mt-4 mb-2">      {/* margin-top: 1rem, margin-bottom: 0.5rem */}
<div className="-ml-2">          {/* negative margin-left: -0.5rem */}
<div className="space-y-4">      {/* adds vertical spacing between children */}
<div className="space-x-2">      {/* adds horizontal spacing between children */}
```

#### Sizing

```tsx
/* === Width === */
<div className="w-full">         {/* width: 100% */}
<div className="w-64">           {/* width: 16rem (256px) */}
<div className="w-1/2">          {/* width: 50% */}
<div className="w-fit">          {/* width: fit-content */}
<div className="w-screen">       {/* width: 100vw */}
<div className="min-w-0">        {/* min-width: 0 (prevents flex overflow) */}

/* === Height === */
<div className="h-full">         {/* height: 100% */}
<div className="h-32">           {/* height: 8rem (128px) */}
<div className="h-screen">       {/* height: 100vh */}
<div className="min-h-[400px]">  {/* min-height: 400px (arbitrary value) */}
<div className="max-h-96 overflow-y-auto"> {/* max-height + scroll */}
```

#### Typography

```tsx
/* === Font Size === */
<p className="text-sm">          {/* font-size: 0.875rem (14px) */}
<p className="text-base">        {/* font-size: 1rem (16px) */}
<p className="text-lg">          {/* font-size: 1.125rem (18px) */}
<p className="text-xl">          {/* font-size: 1.25rem (20px) */}
<p className="text-2xl">         {/* font-size: 1.5rem (24px) */}

/* === Font Weight === */
<p className="font-light">       {/* font-weight: 300 */}
<p className="font-normal">      {/* font-weight: 400 */}
<p className="font-medium">      {/* font-weight: 500 */}
<p className="font-semibold">    {/* font-weight: 600 */}
<p className="font-bold">        {/* font-weight: 700 */}

/* === Line Height === */
<p className="leading-none">     {/* line-height: 1 */}
<p className="leading-tight">    {/* line-height: 1.25 */}
<p className="leading-snug">     {/* line-height: 1.375 */}
<p className="leading-normal">   {/* line-height: 1.5 */}
<p className="leading-relaxed">  {/* line-height: 1.625 */}

/* === Text Color === */
<p className="text-gray-900">    {/* dark gray */}
<p className="text-gray-500">    {/* medium gray */}
<p className="text-gray-400">    {/* light gray */}

<p className="text-blue-600">    {/* blue */}

/* === Text Alignment === */
<p className="text-left">        {/* text-align: left */}
<p className="text-center">      {/* text-align: center */}
<p className="text-right">       {/* text-align: right */}

/* === Text Decoration === */
<p className="underline underline-offset-4 decoration-blue-500">
  {/* underlined text with offset and colored underline */}
</p>
```

#### Backgrounds & Borders

```tsx
/* === Background Color === */
<div className="bg-white">       {/* background-color: white */}
<div className="bg-gray-50">     {/* very light gray */}
<div className="bg-gray-900">    {/* very dark gray / near black */}
<div className="bg-blue-600">    {/* blue background */}
<div className="bg-transparent"> {/* transparent */}

/* === Background with opacity (v4: use CSS color syntax) */
<div className="bg-blue-600/10"> {/* blue background at 10% opacity */}
<div className="bg-black/50">    {/* black background at 50% opacity */}

/* === Border === */
<div className="border">         {/* border: 1px solid current color */}
<div className="border-2">       {/* border-width: 2px */}
<div className="border-b">       {/* border-bottom: 1px solid */}
<div className="border-b-2">     {/* border-bottom-width: 2px */}
<div className="border-gray-200">{/* border-color: gray-200 */}
<div className="border-red-500"> {/* border-color: red-500 */}

/* === Border Radius === */
<div className="rounded">        {/* border-radius: 0.25rem */}

```

#### Effects

```tsx
/* === Shadow === */
<div className="shadow-sm">      {/* small shadow */}
<div className="shadow">         {/* default shadow */}
```

## Chapter 3: Responsive Design

<Badge variant="green">Junior</Badge>

### 3.1 Mobile-First Breakpoints

Tailwind v4 uses a **mobile-first** approach. You design for mobile by default, then add breakpoints for larger screens.

```tsx
/* Breakpoint prefixes (mobile-first):
   (none)  = 0px+      (default / mobile)
   sm:     = 40rem+    (640px)
   md:     = 48rem+    (768px)
   lg:     = 64rem+    (1024px)
   xl:     = 80rem+    (1280px)
   2xl:    = 96rem+    (1536px)
*/

<div className="w-full sm:w-1/2 lg:w-1/3">
  {/* Mobile: full width, Tablet: half, Desktop: third */}
</div>

<h1 className="text-xl sm:text-2xl md:text-3xl lg:text-4xl">
  {/* Responsive font sizes */}
</h1>

<div className="p-4 sm:p-6 md:p-8 lg:p-12">
  {/* Responsive padding */}
</div>
```

### 3.2 Hidden/Visible Patterns

```tsx
{/* Show on desktop, hide on mobile */}
<nav className="hidden md:flex">
  Desktop navigation links
</nav>

{/* Show on mobile, hide on desktop */}
<button className="md:hidden">
  Mobile menu button
</button>

{/* Different layouts per breakpoint */}
<div className="flex flex-col md:flex-row">
  <aside className="w-full md:w-64">Sidebar</aside>
  <main className="flex-1">Main content</main>
</div>
```

### 3.3 Container Queries (Built-in v4!)

Tailwind v4 includes **container queries** out of the box — no plugin needed:

```tsx
/* Mark an element as a container */
<div className="@container">
  {/* Child uses container query: @sm means 24rem container width */}
  <div className="grid grid-cols-1 @sm:grid-cols-2 @lg:grid-cols-3 gap-4">
    <Card />
    <Card />
    <Card />
  </div>
</div>

/* Named containers for precise control */
<div className="@container/card">
  <div className="flex flex-col @sm/card:flex-row">
    <img className="w-full @sm/card:w-1/3" src="..." />
    <div className="flex-1">Content</div>
  </div>
</div>

/* Container size breakpoints:
   @xs  = 20rem  (320px)
   @sm  = 24rem  (384px)
   @md  = 28rem  (448px)
   @lg  = 32rem  (512px)
   @xl  = 36rem  (576px)
   @2xl = 42rem  (672px)
   @3xl = 48rem  (768px)
   @4xl = 56rem  (896px)
   @5xl = 64rem  (1024px)
   @6xl = 72rem  (1152px)
   @7xl = 80rem  (1280px)
*/
```

```tsx
export async function ProductList() {
```

### 5.2 Client Components with Interactive States

When you need interactivity, use Client Components with Tailwind:

```tsx
'use client';

import { useState } from 'react';

export function TabPanel() {
  const [activeTab, setActiveTab] = useState('overview');
  const tabs = ['overview', 'features', 'reviews'];

  return (
    <div className="w-full">
      {/* Tab list */}
      <div className="border-b border-gray-200">
        <nav className="-mb-px flex space-x-8" aria-label="Tabs">
          {tabs.map((tab) => (
            <button
              key={tab}
              onClick={() => setActiveTab(tab)}
              className={`
                whitespace-nowrap border-b-2 px-1 py-4 text-sm font-medium
                transition-colors duration-200
                ${
                  activeTab === tab
                    ? 'border-blue-500 text-blue-600'
                    : 'border-transparent text-gray-500 hover:border-gray-300 hover:text-gray-700'
                }
              `}
            >
              {tab.charAt(0).toUpperCase() + tab.slice(1)}
            </button>
          ))}
        </nav>
      </div>

      {/* Tab content */}
      <div className="mt-6">
        {activeTab === 'overview' && (
          <div className="prose prose-blue max-w-none">
            <p>Overview content here...</p>
          </div>
        )}
        {activeTab === 'features' && (
          <ul className="space-y-3">
            <li className="flex items-start gap-3">
              <CheckIcon className="h-5 w-5 text-green-500 mt-0.5" />
              <span className="text-gray-700">Feature one description</span>
            </li>
          </ul>
        )}
        {activeTab === 'reviews' && <ReviewList />}
      </div>
    </div>
  );
}
```
