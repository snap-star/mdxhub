## Section 1: Getting Started with React 19

### What is React 19?

React 19 represents a paradigm shift in how we build React applications. Released in late 2024, it introduced several game-changing features:

- **React Compiler** вЂ” Automatic memoization without `useMemo`/`useCallback`
- **Server Components** вЂ” Components that render exclusively on the server
- **Actions** вЂ” First-class support for mutations and form submissions
- **Document Metadata API** вЂ” Native `<title>`, `<meta>` support in components
- **Asset Loading** вЂ” Suspense-integrated `<link>` and `<script>` loading
- **Web Components** вЂ” Full support for custom elements
- **Enhanced Hooks** вЂ” `use()` hook for reading promises, context, and more

### Setting Up a React 19 Project

#### Using Vite (Recommended for Learning)

```bash
# Create a new Vite project with React 19
npm create vite@latest my-react-app -- --template react

# Navigate and install dependencies
cd my-react-app
npm install

# Verify React 19 in package.json
# Should show: "react": "^19.0.0"
```

```json
// package.json
```

### Your First React 19 Component

```jsx
// src/App.jsx вЂ” React 19 entry point
function App() {
  return (
    <div className="app">
      <header>
        <h1>Welcome to React 19</h1>
        <p>Server Components, Actions, and more!</p>
      </header>
    </div>
  );
}

export default App;
```

```jsx
// src/main.jsx вЂ” Mounting the app
import { StrictMode } from 'react';
import { createRoot } from 'react-dom/client';
import App from './App';
import './index.css';

// React 19 uses createRoot (same API as React 18)
const root = createRoot(document.getElementById('root'));

root.render(
  <StrictMode>
    <App />
  </StrictMode>
);
```

### Key Differences from React 18

| Feature | React 18 | React 19 |
|---------|----------|----------|
| Memoization | Manual (`useMemo`, `useCallback`) | Automatic with React Compiler |
| Forms | Controlled components + `onSubmit` | Server Actions + `useActionState` |
| Document head | Third-party library (`react-helmet`) | Native `<title>`/`<meta>` support |
| Data fetching | `useEffect` + fetch | `use()` hook + Suspense |
| Async operations | Complex promise handling | `use()` + Actions |
