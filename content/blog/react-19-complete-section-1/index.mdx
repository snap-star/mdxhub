## Section 1: Getting Started with React 19

### What is React 19?

React 19 represents a paradigm shift in how we build React applications. Released in late 2024, it introduced several game-changing features:

- **React Compiler** — Automatic memoization without `useMemo`/`useCallback`
- **Server Components** — Components that render exclusively on the server
- **Actions** — First-class support for mutations and form submissions
- **Document Metadata API** — Native `<title>`, `<meta>` support in components
- **Asset Loading** — Suspense-integrated `<link>` and `<script>` loading
- **Web Components** — Full support for custom elements
- **Enhanced Hooks** — `use()` hook for reading promises, context, and more

### Setting Up a React 19 Project

#### Using Vite (Recommended for Learning)

```bash
# Create a new Vite project with React 19
npm create vite@latest my-react-app -- --template react

# Navigate and install dependencies
cd my-react-app
npm install

# Verify React 19 in package.json
# Should show: "react": "^19.0.0"
```

```json
// package.json
{
  "name": "my-react-app",
  "private": true,
  "version": "0.0.0",
  "type": "module",
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview"
  },
  "dependencies": {
    "react": "^19.0.0",
    "react-dom": "^19.0.0"
  },
  "devDependencies": {
    "@types/react": "^19.0.0",
    "@types/react-dom": "^19.0.0",
    "@vitejs/plugin-react": "^4.0.0",
    "vite": "^6.0.0"
  }
}
```

#### Using Next.js 15 (Full-Stack Framework)

```bash
# Create a Next.js 15 project (includes React 19)
npx create-next-app@latest my-next-app --typescript --tailwind --app

# Next.js 15 uses React 19 Server Components by default
```

### Your First React 19 Component

```jsx
// src/App.jsx — React 19 entry point
function App() {
  return (
    <div className="app">
      <header>
        <h1>Welcome to React 19</h1>
        <p>Server Components, Actions, and more!</p>
      </header>
    </div>
  );
}

export default App;
```

```jsx
// src/main.jsx — Mounting the app
import { StrictMode } from 'react';
import { createRoot } from 'react-dom/client';
import App from './App';
import './index.css';

// React 19 uses createRoot (same API as React 18)
const root = createRoot(document.getElementById('root'));

root.render(
  <StrictMode>
    <App />
  </StrictMode>
);
```

### Key Differences from React 18

| Feature | React 18 | React 19 |
|---------|----------|----------|
| Memoization | Manual (`useMemo`, `useCallback`) | Automatic with React Compiler |
| Forms | Controlled components + `onSubmit` | Server Actions + `useActionState` |
| Document head | Third-party library (`react-helmet`) | Native `<title>`/`<meta>` support |
| Data fetching | `useEffect` + fetch | `use()` hook + Suspense |
| Async operations | Complex promise handling | `use()` + Actions |
