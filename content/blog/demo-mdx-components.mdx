## 2. Tooltip Component

Hover or focus any underlined text to reveal extra information.

### Basic Usage

Wrap any text with a `<Tooltip>` to show additional context on hover:

<Tooltip content="Server Components that run exclusively on the server">RSC</Tooltip> enables efficient data fetching and smaller client bundles.

<Tooltip content="A build-time tool that removes dead code">Tree-shaking</Tooltip> is handled automatically by Vite.

<Tooltip content="React's built-in mechanism for tracking side effects">useEffect</Tooltip> runs after every render by default.

```mdx
<Tooltip content="Server Components that run exclusively on the server">RSC</Tooltip>
enables efficient data fetching and smaller client bundles.
```

### Placement Control

Use the `side` prop to control where the tooltip appears:

<Tooltip content="Appears above (default)" side="top">Top</Tooltip>
<Tooltip content="Appears below" side="bottom">Bottom</Tooltip>
<Tooltip content="Appears to the left" side="left">Left</Tooltip>
<Tooltip content="Appears to the right" side="right">Right</Tooltip>

```mdx
<Tooltip content="Appears above" side="top">Top</Tooltip>
<Tooltip content="Appears below" side="bottom">Bottom</Tooltip>
```

### With Badges

## 3. Image Lightbox Viewer

Click any image in a blog post to open it in a full-screen lightbox viewer. This is already wired up globally — no extra work needed.

### What happens when you click an image

1. The image opens in a **full-screen overlay** with a dark backdrop
2. Use the **toolbar at the bottom** to zoom in/out and rotate
3. The **percentage indicator** shows current zoom level
4. Press **Escape** or click the **backdrop/close button** to dismiss
5. **Alt text** (if present) is shown as a caption at the bottom

### Try it

The cover image at the top of this post is clickable — and so are the images below:

```mdx
![Unsplash abstract](https://images.unsplash.com/photo-1550745165-9bc0b252726f?q=80&w=2670&auto=format&fit=crop)
```
![Unsplash abstract](https://images.unsplash.com/photo-1550745165-9bc0b252726f?q=80&w=2670&auto=format&fit=crop)

### Zoom Controls

| Action | Control |
| :--- | :--- |
| Zoom in | Click the **+** button or scroll on the image area |
| Zoom out | Click the **−** button |
| Rotate | Click the **↻** button (90° clockwise each click) |
| Close | Press **Escape** or click outside the image |
| Reset | Zoom resets automatically when opening a new image |

<Callout type="tip" title="Pro Tip">
The lightbox works with **any image** rendered through the standard MDX `img` component — including images in docs pages and the cover image in the post header (when it uses the img element).
</Callout>

## 5. Steps Component

The `<Steps>` component renders a numbered step-by-step guide with connecting lines. Each step is wrapped in a `<Step title="...">` sub-component.

### Basic Example

<Steps>

<div>

Decide what you want to write. Organize your `.mdx` files in the `content/blog/`
or `content/docs/` directories. Nested folders are supported.

</div>
</Step>
<Step title="Write Your MDX">
<div>

Use standard Markdown syntax enhanced with React components. All the components
on this page are available globally — no imports needed.

</div>
</Step>
<Step title="Preview Locally">
<div>

Run `npm run dev` and navigate to `http://localhost:3000`. Vite's HMR updates
the page instantly as you edit your content.

</div>
</Step>
<Step title="Build &amp; Deploy">
<div>

Run `npm run build` to produce a static `dist/` folder, then deploy to Vercel,
Netlify, or any static hosting provider.

</div>
</Step>
</Steps>

## 8. Quick Reference

| Component | Key Prop | Values | Description |
| :--- | :--- | :---: | :--- |
| **Badge** | `variant` | `tip`, `warning`, `blue`, `emerald`, etc. | Colorful pill for labels and status |
| | `icon` | `check`, `zap`, `star`, `rocket`, etc. | Optional icon from 40+ icon map |
| **Tooltip** | `content` | string | Floating label on hover/focus |
| | `side` | `top`, `bottom`, `left`, `right` | Placement control |
| **Tabs** | `variant` | `underline`, `pills` | Tabbed content panels |
| | `defaultIndex` | number (0-based) | Initial active tab |
| **Steps** | `children` | Step[] | Numbered step-by-step container |
| **Step** | `title` | string | Step heading text |
| | `icon` | string | Optional icon name |
| **Mermaid** | `chart` | string | Diagram definition text |
| | `caption` | string | Optional caption below diagram |
| **Image** | *(all)* | — | Click any image → opens lightbox |
