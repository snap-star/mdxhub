This section covers the most common issues you may encounter while working with this platform, from local development hiccups to build failures and content not appearing as expected. Use the sidebar to navigate directly to the relevant category.

<Callout type="tip" title="Quick Self-Check">
Before diving into specific issues, always try these steps first:

1. `npm install` — ensure all dependencies are up to date
2. Delete the `dist/` folder and restart `npm run dev`
3. Check the browser console for specific error messages
</Callout>

## How to Report a Bug

If you cannot resolve an issue using this guide, open an issue on the project's GitHub repository. Include:

- The **exact error message** from the terminal or browser console
- Your **Node.js version** (`node -v`) and **npm version** (`npm -v`)
- The relevant `.mdx` file content (with frontmatter)
- Steps to reproduce the problem
