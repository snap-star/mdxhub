Welcome to **ReactMDX** вЂ” a production-ready, zero-configuration content platform that combines the power of React 19, MDX, Vite, and TailwindCSS v4 into a stunning and developer-friendly writing experience.

## What is ReactMDX?

ReactMDX is a Single Page Application that gives you the best of both worlds:

- **Markdown simplicity** вЂ” write content naturally without fighting a CMS
- **React power** вЂ” embed interactive components, videos, callouts, and charts anywhere in your prose
- **Zero routing config** вЂ” drop an `.mdx` or `.md` file into the right folder and it instantly appears, auto-discovered by Vite's `import.meta.glob`
- **Nested routes supported** вЂ” blog and docs support nested folders, and `index.mdx` inside a folder maps to the parent slug
- **Relative asset loading** вЂ” images inside `content/blog/**` and `content/docs/**` can be referenced with relative paths in MDX

## Core Architecture

This project is built on the latest 2026 web standards:

1. **React 19** вЂ” concurrent features, `use()`, and the new compiler
2. **Vite 8** вЂ” lightning-fast HMR and optimized production builds
3. **React Router v8** вЂ” file-based-style nested SPA routing
4. **TailwindCSS v4** вЂ” CSS-first config with OKLCH color spaces
5. **Zustand v5** вЂ” minimal, boilerplate-free global state
6. **Shiki v4** вЂ” build-time syntax highlighting with full VS Code theme support
7. **Framer Motion 12** вЂ” production-grade animations and page transitions
